# SkillWallet - Personal Finance Management System

## Project Overview

SkillWallet is a full-stack personal finance management application that helps users track income, expenses, savings, and financial activities. The application provides secure authentication, transaction management, analytics dashboards, and report generation.

## Features

### User Authentication

* User Registration
* User Login
* JWT Authentication
* Secure Logout

### Transaction Management

* Add Transactions
* Edit Transactions
* Delete Transactions
* Search Transactions
* Filter Transactions

### Dashboard Analytics

* Current Balance
* Total Income
* Total Expenses
* Total Transactions
* Highest Income
* Highest Expense
* Recent Transactions
* Monthly Summary

### Reports

* Export Transactions to PDF
* Export Transactions to CSV

### Data Visualization

* Pie Chart
* Bar Chart
* Financial Analytics Dashboard

### Database

* MongoDB Integration
* CRUD Operations

## Technologies Used
